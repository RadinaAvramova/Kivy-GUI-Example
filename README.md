# Kivy GUI Example
An example GUI created in python using the module kivy. This is a very basic login system where you can create accounts and login.

# Requirements

Linux and MacOS:
- Python 3.5+
- kivy

Windows:
- Python 3.5+
- pygame
- docutils 
- pygments 
- pypiwin32 
- kivy.deps.sdl2 
- kivy.deps.glew
- kivy.deps.gstreamer
- kivy.deps.angle
- kivy
